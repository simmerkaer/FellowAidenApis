☕ fellow-aiden

A JavaScript/TypeScript client for the Fellow Aiden API. Sign in below to see it pull your brewer's profiles & schedules — live.

🔒 This demo is read-only — it lists your profiles and schedules and changes nothing.

🙈 Your credentials are sent to Fellow to log in and are never stored, logged, or shared by this page.

🌐 Browsers can't call Fellow's API directly (no CORS), so requests pass through a thin relay. How it works ↓

      The code running this page

      import { FellowAiden } from 'fellow-aiden';

const aiden = await FellowAiden.create({ email, password });

aiden.getDisplayName();      // "Kitchen Aiden"
await aiden.getProfiles();   // [{ id, title, ... }]
await aiden.getSchedules();  // [{ id, ... }]

      How it works

      The Fellow API is a mobile-app backend with no CORS support, so a web page can't call it directly. This demo runs the fellow-aiden library in your browser and routes its requests through a small Cloudflare Worker that forwards them to Fellow and adds the missing CORS headers. The relay only passes requests through — it stores nothing.
